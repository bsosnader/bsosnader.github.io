About Me

Hi there! I've been a self-taught web designer and developer since I was twelve. I can make you a portfolio, homepage, blog, or any other simple site you require. I specialize in creating beautiful, dynamic web pages optimized for both mobile, and desktop. I'm flexible, and if you want a specific feature, I'll try my best to implement it. If you need a content management system, or prefer a certain type of website, I'm happy to cater to your needs.

HTML

HTML is the foundation for any website and forms the most basic building blocks of a site. It provides the framework for the content of the site to be built around. I use HTML5 with my websites to take advantage of the newest features and optizations it provides. If you need legacy support for older browsers, I can provide that as well.

CSS

CSS provides the beauty to make the page look like something other than plain old text. CSS is the most powerful tool in web design because of how adaptable it is. I use SCSS and Compass to make easily editable and powerful CSS.

Javascript

Javascript is what adds that extra pop to a website. Without it, many more complex features of websites would not be able to be integrated. I use Javascript and JQuery to provide robust finishing touches to extra features that really make a site stand out.
